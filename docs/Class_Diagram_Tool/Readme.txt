To use this toool you need to have pylint installed (can be done through PyCharm) and Graphviz installed (https://graphviz.gitlab.io/_pages/Download/Download_windows.html).

You will also need to set the following environment variables:

Add Python\Scripts to path
	C:\Users\sgould\AppData\Roaming\Python\Scripts
	C:\python27\ArcGIS10.3\Scripts
Add graphviz to path	
	C:\Program Files (x86)\Graphviz2.38\bin
